## 🎈텐서 초기화

### 방법 0 : 직접 초기화하기
shape는 기본적으로 튜플로 적는다. 리스트도 가능하다.<br>
(**row, column**)이나
(**dimension, row, column**)으로 구성
```
shape = (3,4,)
x_ones = torch.ones(shape)
x_zeros = torch.zeros(shape)
x_rand = torch.rand(shape)
  #0~1사이 랜덤 float

  #1~99까지 랜덤 int
```
